# Product Requirements Document (PRD)

## Problem
Users Often lose Use full links saved in browser. They are not organized, portable, or easyly accesible on other devices.

## Solution
LinkSaver is a simple web app that allow users to save, tag, and view link from anyware. It stores link in Vercel for easy access.

## Users
- Traveler
- Foreigners 
- Genaral Users

## Key Features
- Save Link With Title and Optional Tag
- View all save links in real time
- Store Links In cloud (Vercel)
